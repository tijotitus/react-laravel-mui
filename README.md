
## Laravel React with Material UI Minimal template

## Database - Sqlite
## How to run

Please run below commands,

- composer install
- npm install
- php artisan migrate
- php artisan db:seed --class=UsersTableSeeder
- php artisan key:generate ( note : rename env.example to .env ) 
- npm run dev
- php artisan serve
